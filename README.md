# Portfolio

Personal art portfolio with tabs for Bio, Painting, Sculpture, and Drawing.

## Setup

1. Add your artwork images to the `img/` folder
2. Place `biophoto.jpg` in the root for the bio tab
3. Open `index.html` in your browser

## Image Naming

- `img/painting-1.jpg` through `img/painting-6.jpg`
- `img/sculpture-1.jpg` through `img/sculpture-6.jpg`
- `img/drawing-1.jpg` through `img/drawing-9.jpg`
- `biophoto.jpg` (root)
